
# AI Chatbot Assistant Using LLMs and RAG

A production-style AI chatbot assistant built with React, FastAPI, LangChain, and Gemini/OpenAI APIs.

## Features
- AI Chatbot
- JWT Authentication
- Resume ATS Analyzer
- PDF Q&A
- Chat History
- REST APIs
- Responsive UI

## Tech Stack
Frontend: React + Tailwind CSS
Backend: FastAPI + Python
Database: MongoDB
AI: Gemini/OpenAI APIs + LangChain

## Run Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

## Run Frontend
```bash
cd frontend
npm install
npm start
```
